# 滑动窗口验证策略讨论记录

## 背景

用户希望验证一个交易指导方法的可行性：**用历史数据优化出的参数，能否有效指导未来短期走势？**

---

## 一、核心思路

### 问题本质

由于无法获取未来的数据，我们希望通过历史数据来验证策略的有效性。但不是为了回测整个历史，而是模拟"在当时用当时的数据做决策"的场景。

### 验证方法

使用**滑动窗口**的方式，每5根K线进行一次计算，用前300条数据优化参数，然后"预测"接下来5根的走势，逐步滑动验证。

---

## 二、数据获取策略

### OKEX API限制

- 每次最多请求300条K线数据
- 需要调用2次接口获取600条数据

### 数据分段

```
600条数据索引: [0, 1, 2, ..., 299, 300, 301, ..., 599]
                    │                    │
                    └─ 第1次请求 ─┘     └─ 第2次请求 ─┘
                    (历史300条)         (最新300条)
```

### 获取逻辑

```javascript
// 第1次请求：获取最新的300条（OKX返回新→旧顺序）
const candles1 = await fetchOKXData(pair, timeframe, 300);

// 第2次请求：获取更早的300条（使用before参数）
const earliestTime = candles1[candles1.length - 1][0];
const candles2 = await fetchOKXDataBefore(pair, timeframe, 300, earliestTime);

// 合并数据：candles1(较新) + candles2(较旧)，保持新→旧顺序
// 这样 parseOKXCandles 的 reverse() 会将其转为旧→新顺序
const allCandles = [...candles1, ...candles2];
```

**注意**：OKX API返回的数据格式为 `[timestamp, open, high, low, close, vol, volCcy]`，经过 `parseOKXCandles` 处理后变为 `[open, close, low, high, volume]` 格式。

---

## 三、滑动窗口计算逻辑

### 窗口定义

```
轮次    优化窗口(300条)         预测/验证窗口(5条)
─────────────────────────────────────────────────
第1轮   [0-299]                [300-304]  
第2轮   [5-304]                [305-309]
第3轮   [10-309]               [310-314]
...     ...                    ...
第60轮  [295-594]              [595-599]
```

### 每轮计算步骤

```
1. 取当前优化窗口的300条数据
        ↓
2. 对这300条进行参数优化
   - 测试所有快线(4-20) × 慢线(20,30,40,50,60,80,100)组合
   - 找出收益率最高的(fast, slow)参数对
        ↓
3. 用最优参数计算优化窗口最后一根K线的信号
   - 金叉 → 预测"做多"
   - 死叉 → 预测"做空"
   - 无信号 → 预测"观望"
        ↓
4. 验证预测：看接下来5根K线的实际走势
   - 计算这5根的收益/亏损
   - 记录预测是否正确
        ↓
5. 窗口向后滑动5根K线，重复步骤1-4
```

## 四、核心算法实现

### 1. 获取600条数据

```javascript
async function fetch600Candles(pair, timeframe) {
    // 第1次：获取最新300条
    const candles1 = await fetchOKXData(pair, timeframe, 300);
    
    // 获取最早一根的时间戳，用于第2次请求
    const earliestTime = candles1[candles1.length - 1][0];
    
    // 第2次：获取更早的300条
    const candles2 = await fetchOKXDataBefore(pair, timeframe, 300, earliestTime);
    
    // 合并并排序（时间升序）
    const allCandles = [...candles2, ...candles1];
    allCandles.sort((a, b) => parseInt(a[0]) - parseInt(b[0]));
    
    return allCandles;
}
```

### 2. 滑动窗口验证

```javascript
async function runSlidingWindowValidation(pair, timeframe) {
    // 获取600条数据
    const allData = await fetch600Candles(pair, timeframe);
    const parsed = parseOKXCandles(allData);
    const { dates, data } = parsed;
    
    const windowSize = 300;  // 优化窗口大小
    const stepSize = 5;      // 每次滑动5根
    const predictSize = 5;   // 预测未来5根
    
    const results = [];
    
    // 从第0根开始滑动，直到用完所有数据
    for (let start = 0; start <= data.length - windowSize - predictSize; start += stepSize) {
        const optimizeEnd = start + windowSize;      // 优化窗口结束
        const predictEnd = optimizeEnd + predictSize; // 预测窗口结束
        
        // 1. 取优化窗口数据
        const optimizeDates = dates.slice(start, optimizeEnd);
        const optimizeData = data.slice(start, optimizeEnd);
        
        // 2. 参数优化
        const bestParams = optimizeParameters(optimizeDates, optimizeData);
        // 返回: { maFast, maSlow, totalReturn, winRate, maxDrawdown }
        
        // 3. 用最优参数计算最后一根的信号
        const signal = calculateSignalAtIndex(
            optimizeData, 
            bestParams.maFast, 
            bestParams.maSlow,
            optimizeEnd - 1  // 最后一根索引
        );
        // 返回: { trendSignal: 'long'|'short'|'neutral', signalAge, currentAdx }
        
        // 4. 取预测窗口数据验证
        const predictDates = dates.slice(optimizeEnd, predictEnd);
        const predictData = data.slice(optimizeEnd, predictEnd);
        
        // 5. 验证预测结果
        const validation = validatePrediction(
            predictData,
            signal.trendSignal,
            optimizeData[optimizeEnd - 1][1] // 优化窗口最后一根收盘价作为入场价
        );
        // 返回: { actualReturn, predictionCorrect, maxProfit, maxLoss }
        
        results.push({
            round: results.length + 1,
            startIndex: start,
            endIndex: optimizeEnd - 1,
            predictStart: optimizeEnd,
            predictEnd: predictEnd - 1,
            params: bestParams,
            signal: signal,
            validation: validation,
            dates: {
                optimizeStart: dates[start],
                optimizeEnd: dates[optimizeEnd - 1],
                predictStart: dates[optimizeEnd],
                predictEnd: dates[predictEnd - 1]
            }
        });
    }
    
    return results;
}
```

### 3. 参数优化函数

```javascript
function optimizeParameters(dates, data) {
    const fastRange = Array.from({length: 17}, (_, i) => i + 4); // 4-20
    const slowRange = [20, 30, 40, 50, 60, 80, 100];
    
    // 使用默认的其他参数
    const adxThreshold = 5;
    const stopLoss = 0.10;
    const takeProfit = 1.0;
    const initialCapital = 1000;
    const stakeAmount = 1000;
    const enableShort = true;
    
    let bestResult = null;
    
    for (const fast of fastRange) {
        for (const slow of slowRange) {
            if (fast >= slow) continue;
            
            const result = runBacktestWithParams(
                dates, data, fast, slow, 
                adxThreshold, stopLoss, takeProfit, 
                initialCapital, stakeAmount, enableShort
            );
            
            if (!bestResult || result.totalReturn > bestResult.totalReturn) {
                bestResult = { ...result, maFast: fast, maSlow: slow };
            }
        }
    }
    
    return bestResult;
}
```

### 4. 计算指定索引的信号

```javascript
function calculateSignalAtIndex(data, maFast, maSlow, index) {
    const maFastValues = calculateMA(data, maFast);
    const maSlowValues = calculateMA(data, maSlow);
    const adxValues = calculateADX(data, 14);
    
    const currFast = parseFloat(maFastValues[index]);
    const currSlow = parseFloat(maSlowValues[index]);
    const prevFast = parseFloat(maFastValues[index - 1]);
    const prevSlow = parseFloat(maSlowValues[index - 1]);
    const currentAdx = adxValues[index];
    
    let trendSignal = 'neutral';
    let signalAge = 0;
    
    if (currFast > currSlow) {
        trendSignal = 'long';
        // 往前找金叉点
        for (let i = index; i > 0; i--) {
            const f = parseFloat(maFastValues[i]);
            const s = parseFloat(maSlowValues[i]);
            const pf = parseFloat(maFastValues[i-1]);
            const ps = parseFloat(maSlowValues[i-1]);
            if (pf <= ps && f > s) {
                signalAge = index - i;
                break;
            }
        }
    } else if (currFast < currSlow) {
        trendSignal = 'short';
        // 往前找死叉点
        for (let i = index; i > 0; i--) {
            const f = parseFloat(maFastValues[i]);
            const s = parseFloat(maSlowValues[i]);
            const pf = parseFloat(maFastValues[i-1]);
            const ps = parseFloat(maSlowValues[i-1]);
            if (pf >= ps && f < s) {
                signalAge = index - i;
                break;
            }
        }
    }
    
    return { trendSignal, signalAge, currentAdx };
}
```

### 5. 验证预测结果

```javascript
function validatePrediction(predictData, predictedSignal, entryPrice) {
    const entry = parseFloat(entryPrice);
    let maxProfit = 0;
    let maxLoss = 0;
    let finalReturn = 0;
    
    // 遍历预测窗口的5根K线
    for (let i = 0; i < predictData.length; i++) {
        const high = parseFloat(predictData[i][3]); // 最高价
        const low = parseFloat(predictData[i][2]);  // 最低价
        const close = parseFloat(predictData[i][1]); // 收盘价
        
        if (predictedSignal === 'long') {
            // 做多：看涨幅
            const profit = (high - entry) / entry;
            const loss = (low - entry) / entry;
            maxProfit = Math.max(maxProfit, profit);
            maxLoss = Math.min(maxLoss, loss);
            finalReturn = (close - entry) / entry;
        } else if (predictedSignal === 'short') {
            // 做空：看跌幅
            const profit = (entry - low) / entry;
            const loss = (entry - high) / entry;
            maxProfit = Math.max(maxProfit, profit);
            maxLoss = Math.min(maxLoss, loss);
            finalReturn = (entry - close) / entry;
        } else {
            // 观望：看绝对波动
            const up = (high - entry) / entry;
            const down = (low - entry) / entry;
            maxProfit = Math.max(maxProfit, up);
            maxLoss = Math.min(maxLoss, down);
            finalReturn = 0; // 观望不参与
        }
    }
    
    // 判断预测是否正确
    let predictionCorrect = false;
    if (predictedSignal === 'long' && finalReturn > 0) predictionCorrect = true;
    if (predictedSignal === 'short' && finalReturn > 0) predictionCorrect = true;
    if (predictedSignal === 'neutral') predictionCorrect = true; // 观望不算错
    
    return {
        actualReturn: finalReturn,
        predictionCorrect,
        maxProfit,
        maxLoss,
        holdingPeriod: predictData.length
    };
}
```

## 五、实际仓位模拟系统

### 核心概念

不再是简单的"预测对错"，而是**模拟真实的开仓-持仓-平仓过程**，计算实际的资金变化。

### 仓位状态定义

```
position: 0   // 空仓
position: 1   // 做多持仓
position: -1  // 做空持仓

entryPrice: 0     // 开仓价格
entryIndex: 0     // 开仓K线索引
entryTime: ''     // 开仓时间
currentCapital: 1000  // 当前资金
```

### 每轮交易逻辑

```
第N轮开始:
    1. 根据优化窗口的信号，决定操作
       - 信号=做多 & 空仓 → 开多仓
       - 信号=做空 & 空仓 → 开空仓
       - 信号=观望 & 空仓 → 继续观望

    2. 如果已有持仓:
       - 信号方向与持仓一致 → 继续持有（不操作）
       - 信号方向与持仓相反 → 平仓（标记"信号反向"），等待下一轮再开新仓
       - 触发止损/止盈 → 平仓

    3. 在预测窗口的5根K线中:
       - 跟踪持仓盈亏
       - 检查止损/止盈条件
       - 记录资金曲线变化

    4. 第5根K线结束时:
       - 如果信号方向与持仓一致 → 继续持有到下一轮
       - 如果信号方向与持仓相反 → 已在第1根K线平仓
```

**关键改进**：与早期版本不同，当前版本不会在每轮结束时强制平仓。只有当信号反向或触发止损/止盈时才会平仓，相同方向时保持持仓，减少不必要的交易摩擦。

### 完整仓位管理代码

```javascript
function simulateTradingWithPosition(allData, dates, config) {
    const {
        windowSize = 300,      // 优化窗口
        stepSize = 5,          // 滑动步长
        predictSize = 5,       // 预测窗口
        initialCapital = 1000, // 初始资金
        stakeAmount = 1000,    // 单次仓位
        stopLoss = 0.10,       // 止损10%
        takeProfit = 0.20,     // 止盈20%
        enableShort = true,    // 允许做空
        closeOnNeutral = false // 观望时是否平仓
    } = config;
    
    let capital = initialCapital;
    let position = 0;          // 0:空仓, 1:多仓, -1:空仓
    let entryPrice = 0;
    let entryIndex = 0;
    let entryTime = '';
    
    const trades = [];         // 交易记录
    const equityCurve = [{     // 资金曲线
        index: 0,
        time: dates[0],
        capital: initialCapital,
        position: 0
    }];
    const positionHistory = []; // 仓位历史
    
    // 从第300根开始滑动
    for (let start = windowSize; start <= allData.length - predictSize; start += stepSize) {
        const optimizeStart = start - windowSize;
        const optimizeEnd = start;
        const predictEnd = start + predictSize;
        
        // 1. 取优化窗口数据并优化参数
        const optimizeDates = dates.slice(optimizeStart, optimizeEnd);
        const optimizeData = allData.slice(optimizeStart, optimizeEnd);
        const bestParams = optimizeParameters(optimizeDates, optimizeData);
        
        // 2. 用最优参数计算当前信号
        const signal = calculateSignalAtIndex(
            optimizeData, 
            bestParams.maFast, 
            bestParams.maSlow,
            windowSize - 1
        );
        
        // 3. 获取预测窗口数据
        const predictData = allData.slice(optimizeEnd, predictEnd);
        const predictDates = dates.slice(optimizeEnd, predictEnd);
        
        // 4. 执行交易逻辑
        for (let i = 0; i < predictData.length; i++) {
            const currentIdx = optimizeEnd + i;
            const close = parseFloat(predictData[i][1]);
            const high = parseFloat(predictData[i][3]);
            const low = parseFloat(predictData[i][2]);
            const currentTime = predictDates[i];
            
            // 计算当前持仓的浮动盈亏
            let unrealizedPnl = 0;
            if (position === 1) {
                unrealizedPnl = (close - entryPrice) / entryPrice;
            } else if (position === -1) {
                unrealizedPnl = (entryPrice - close) / entryPrice;
            }
            
            // 检查止损/止盈
            let shouldClose = false;
            let closeReason = '';
            
            if (position !== 0) {
                // 检查止损
                if (unrealizedPnl <= -stopLoss) {
                    shouldClose = true;
                    closeReason = '止损';
                }
                // 检查止盈
                else if (unrealizedPnl >= takeProfit) {
                    shouldClose = true;
                    closeReason = '止盈';
                }
                // 最后一根K线强制平仓
                else if (i === predictData.length - 1) {
                    shouldClose = true;
                    closeReason = '策略调仓';
                }
            }
            
            // 执行平仓
            if (shouldClose && position !== 0) {
                const pnl = position === 1 
                    ? (close - entryPrice) / entryPrice
                    : (entryPrice - close) / entryPrice;
                
                const tradePnl = stakeAmount * pnl;
                capital += tradePnl;
                
                trades.push({
                    entryIndex,
                    exitIndex: currentIdx,
                    entryTime,
                    exitTime: currentTime,
                    entryPrice,
                    exitPrice: close,
                    direction: position === 1 ? 'long' : 'short',
                    pnl: pnl,
                    pnlAmount: tradePnl,
                    closeReason,
                    holdingPeriod: currentIdx - entryIndex
                });
                
                position = 0;
                entryPrice = 0;
            }
            
            // 根据信号开新仓或调仓
            if (position === 0) {
                // 空仓状态下根据信号开仓
                if (signal.trendSignal === 'long') {
                    position = 1;
                    entryPrice = close;
                    entryIndex = currentIdx;
                    entryTime = currentTime;
                } else if (signal.trendSignal === 'short' && enableShort) {
                    position = -1;
                    entryPrice = close;
                    entryIndex = currentIdx;
                    entryTime = currentTime;
                }
            } else {
                // 有持仓时，检查是否需要调仓
                const shouldLong = signal.trendSignal === 'long';
                const shouldShort = signal.trendSignal === 'short';
                
                // 多仓时遇到做空信号 → 平多开空
                if (position === 1 && shouldShort && enableShort) {
                    // 先平多（上面已经处理了平仓）
                    // 再开空
                    position = -1;
                    entryPrice = close;
                    entryIndex = currentIdx;
                    entryTime = currentTime;
                }
                // 空仓时遇到做多信号 → 平空开多
                else if (position === -1 && shouldLong) {
                    // 先平空（上面已经处理了平仓）
                    // 再开多
                    position = 1;
                    entryPrice = close;
                    entryIndex = currentIdx;
                    entryTime = currentTime;
                }
                // 观望信号时是否平仓
                else if (signal.trendSignal === 'neutral' && closeOnNeutral && !shouldClose) {
                    shouldClose = true;
                    closeReason = '观望平仓';
                    // 重新执行平仓逻辑...
                }
            }
            
            // 记录资金曲线和仓位
            const currentEquity = capital + (position !== 0 ? stakeAmount * unrealizedPnl : 0);
            equityCurve.push({
                index: currentIdx,
                time: currentTime,
                capital: currentEquity,
                position,
                unrealizedPnl: position !== 0 ? unrealizedPnl : 0
            });
            
            positionHistory.push({
                index: currentIdx,
                time: currentTime,
                position,
                price: close,
                signal: signal.trendSignal
            });
        }
    }
    
    // 验证结束，如果还有持仓，按最后一根K线价格强制平仓
    if (position !== 0 && positionHistory.length > 0) {
        const lastPosition = positionHistory[positionHistory.length - 1];
        const lastClose = lastPosition.close;
        const lastTime = lastPosition.time;
        const lastIdx = lastPosition.index;
        
        const pnl = position === 1
            ? (lastClose - entryPrice) / entryPrice
            : (entryPrice - lastClose) / entryPrice;
        
        const tradePnl = stakeAmount * pnl;
        capital += tradePnl;
        
        trades.push({
            entryIndex: entryIndex,
            exitIndex: lastIdx,
            entryTime: entryTime,
            exitTime: lastTime,
            entryPrice: entryPrice,
            exitPrice: lastClose,
            direction: position === 1 ? 'long' : 'short',
            pnl: pnl,
            pnlAmount: tradePnl,
            closeReason: '验证结束',  // 标记为验证结束平仓
            holdingPeriod: lastIdx - entryIndex
        });
        
        position = 0;
    }
    
    return {
        finalCapital: capital,
        totalReturn: (capital - initialCapital) / initialCapital,
        trades,
        equityCurve,
        positionHistory
    };
}
```

**注意**：验证结束时强制平仓是为了计算最终收益，实际交易中不会因此产生交易。

## 六、实际交易指标展示

### 核心指标

| 指标 | 说明 | 计算公式 |
|------|------|---------|
| **初始资金** | 起始资金 | 固定值，如1000 USDT |
| **最终资金** | 验证结束后的资金 | 实时计算 |
| **总收益率** | 整体盈亏比例 | (最终-初始)/初始 × 100% |
| **总交易次数** | 完整开平仓次数 | trades.length |
| **盈利次数** | 正收益交易数 | trades.filter(t => t.pnl > 0).length |
| **亏损次数** | 负收益交易数 | trades.filter(t => t.pnl < 0).length |
| **胜率** | 盈利交易占比 | 盈利次数/总交易次数 |
| **平均盈利** | 盈利交易的平均收益率 | 盈利总和/盈利次数 |
| **平均亏损** | 亏损交易的平均收益率 | 亏损总和/亏损次数 |
| **盈亏比** | 平均盈利/平均亏损 | 越大越好，>1.5优秀 |
| **最大回撤** | 资金曲线的最大跌幅 | 峰值到谷值的最大比例 |
| **夏普比率** | 风险调整收益 | (收益率-无风险利率)/波动率 |
| **持仓时间** | 平均持仓K线数 | 总持仓K线/交易次数 |

## 八、统计指标详解

### 预测准确率 vs 实际胜率

| 指标 | 定义 | 区别 |
|------|------|------|
| **预测准确率** | 信号方向与实际走势一致的比例 | 只看方向对不对 |
| **实际胜率** | 盈利交易次数/总交易次数 | 看实际赚没赚钱 |

**示例**：
- 预测做多，实际涨了3% → 预测正确✅，交易盈利✅
- 预测做多，实际涨了0.1% → 预测正确✅，但扣除手续费可能亏损❌
- 预测做多，实际跌了但中途涨了5% → 预测错误❌，但可能止盈盈利✅

### 夏普比率计算

```javascript
function calculateSharpeRatio(equityCurve, riskFreeRate = 0.02) {
    // 计算收益率序列
    const returns = [];
    for (let i = 1; i < equityCurve.length; i++) {
        const r = (equityCurve[i].capital - equityCurve[i-1].capital) / equityCurve[i-1].capital;
        returns.push(r);
    }
    
    // 平均收益率
    const avgReturn = returns.reduce((a, b) => a + b, 0) / returns.length;
    
    // 收益率标准差（波动率）
    const variance = returns.reduce((sum, r) => sum + Math.pow(r - avgReturn, 2), 0) / returns.length;
    const volatility = Math.sqrt(variance);
    
    // 夏普比率
    const sharpeRatio = volatility > 0 ? (avgReturn - riskFreeRate) / volatility : 0;
    
    return sharpeRatio;
}
```

### 最大回撤计算

```javascript
function calculateMaxDrawdown(equityCurve) {
    let maxDrawdown = 0;
    let peak = equityCurve[0].capital;
    let peakIndex = 0;
    let drawdownStart = 0;
    let drawdownEnd = 0;
    
    for (let i = 1; i < equityCurve.length; i++) {
        const capital = equityCurve[i].capital;
        
        if (capital > peak) {
            peak = capital;
            peakIndex = i;
        }
        
        const drawdown = (peak - capital) / peak;
        if (drawdown > maxDrawdown) {
            maxDrawdown = drawdown;
            drawdownStart = peakIndex;
            drawdownEnd = i;
        }
    }
    
    return {
        maxDrawdown,
        drawdownStart,
        drawdownEnd,
        startCapital: equityCurve[drawdownStart].capital,
        endCapital: equityCurve[drawdownEnd].capital
    };
}
```

## 七、方法可行性评估

### 优势

1. **模拟真实场景**：用当时的数据做当时的决策，而非事后诸葛亮
2. **参数自适应**：每轮都重新优化，适应市场变化
3. **可量化验证**：有明确的预测对错判断标准
4. **短期预测**：5根K线的预测比长期预测更可靠

### 局限性

1. **样本量有限**：600条数据只能做约60轮验证
2. **过拟合风险**：300条数据优化可能过拟合
3. **滑点未考虑**：实际交易有滑点和手续费
4. **市场变化**：历史规律不一定适用于未来

### 改进方向

1. **增加数据量**：如果API支持，获取更多历史数据
2. **多品种验证**：在不同交易对上测试，看普适性
3. **组合策略**：结合多个指标，不仅看均线
4. **动态调整**：根据验证结果动态调整参数范围
